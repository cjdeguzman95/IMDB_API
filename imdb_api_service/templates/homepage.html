Welcome to the IMDB API

This is a simple web API that gives you some information from the IMDB database, in JSON format.

To navigate to the different pages, copy and paste the route, inside the parenthesis, onto the web browser.
For example, for all movie data: 127.0.0.1:5000/all-movie-data

Information available:

All data from the database

(/all-movie-data)

All films that grossed over half billion

(/all-films-over-half-a-billion)

Top 10 UK films

(/top-10-uk-films)
